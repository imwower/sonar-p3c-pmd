Look for qualified this usages in the same class.

Examples:

    public class DoNotThrowExceptionInIfElseStatementRule {
        public void test() {
            String abc = "abc";
            if(abc == "abc"){
                System.out.println("abc");
            }else if (abc == "def"){
                System.out.println("def");
            }else {
                throw new Exception();
            }
        }
    }
