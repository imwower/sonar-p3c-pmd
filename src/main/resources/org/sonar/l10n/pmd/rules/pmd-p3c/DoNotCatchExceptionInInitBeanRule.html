Look for qualified this usages in the same class.

Examples:

public class Server {
    public static void main(String[] args) {
        Server server = new Server();
        server.start();
        try {
            while (true) {
                Thread.sleep(Long.MAX_VALUE);
            }
        } catch (Exception e) {
            log.error(server.getClass().getName() + " run error ", e);
        }
    }
}
